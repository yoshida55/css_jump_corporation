# コーポレートサイト（Web 制作会社） コーディング仕様

## 基本設定

| 項目             | 値                                       |
| ---------------- | ---------------------------------------- |
| コンテンツ幅     | 1000px（メインビジュアル・About は全幅） |
| ブレイクポイント | 900px                                    |

---

## 各セクション仕様

### Header

- 右上に「お問い合わせ」ボタンを配置
- SP では「お問い合わせ」ボタンを非表示にする（`display: none`）

### メインビジュアル

- 幅：全幅（100%）
- 高さ：画面下まで（`100vh`）

### セクションタイトル共通

- 英字タイトル + 日本語タイトルの 2 段構成
- 英字タイトルは `letter-spacing` で文字間隔を広げる

### News

- 3 カラムに分割し、間に罫線を引く
- 日付は `<time>` タグを使用する

### About

- 幅：全幅（100%）
- 画像の高さ：400px 固定（`object-fit: cover` で調整）

### Business

- 高さをずらした横並び配置
- SP 時の画像の並び順に注意
  - HTML の記述順 または `order` プロパティで調整を検討

### Company

- 幅：1000px
- 画像を上に重ねて配置（`position` を使用）
- 画像の高さ：400px 固定

---

## レスポンシブ対応

```css
/* PC */
@media (min-width: 901px) {
  /* PC用スタイル */
}

/* SP */
@media (max-width: 900px) {
  /* SP用スタイル */
}
```
